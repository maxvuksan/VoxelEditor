# Voxel Editor

A level editor which prerenders voxel enviroments to images.

![Render Screenshot](/example2.png)
![Render Screenshot](/example3.png)

After rendering, the level can be previewed with a selected palette, the palette is made up of lit colours, in shadow colours and fog/background colour.
